# UNProgress
UNS.py (Runs flask application)
- Create Python Virtual Environment
- Open directory that holds UNS.py
- In command prompt:
    + pip install flask
    + set FLASK_DEBUG=1
    + set FLASK_APP=UNS.py
    + flask run
- Into a browser, paste given local URL as web address
